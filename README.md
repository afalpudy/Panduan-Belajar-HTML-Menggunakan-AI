# Panduan-Belajar-HTML-Menggunakan-AI
Panduan belajar HTML menggunakan AI
1. Pahami Dasar HTML Terlebih Dahulu

HTML digunakan untuk membuat struktur halaman web.

Contoh sederhana:

<!DOCTYPE html>
<html>
<head>
    <title>Website Pertama Saya</title>
</head>
<body>
    <h1>Halo Dunia!</h1>
    <p>Saya sedang belajar HTML.</p>
</body>
</html>

Pelajari tag dasar berikut:

<html>
<head>
<title>
<body>
<h1> sampai <h6>
<p>
<a>
<img>
<div>
<ul> dan <li>
2. Gunakan AI Sebagai Guru Pribadi

Contoh prompt yang bisa kamu berikan ke AI:

Meminta Penjelasan

Jelaskan fungsi tag <div> dalam HTML dengan bahasa sederhana.

Meminta Contoh

Buatkan contoh website profil sederhana menggunakan HTML.

Memperbaiki Error

Berikut kode HTML saya, mengapa gambar tidak muncul?

Lalu tempelkan kodenya.

Meminta Tantangan

Berikan saya latihan HTML tingkat pemula beserta jawabannya.

3. Metode Belajar 80% Praktik

Jangan hanya membaca teori.

Buat proyek kecil:

Biodata diri
Halaman hobi
Portofolio sederhana
Website sekolah
Landing page produk

Karena kamu sedang membuat portofolio, itu sudah langkah yang bagus.

4. Gunakan AI Untuk Review Kode

Contoh:

Tolong cek kode HTML saya. Beri tahu bagian yang kurang rapi dan bagaimana memperbaikinya.

AI bisa membantu menemukan:

Tag yang belum ditutup
Struktur yang kurang baik
Penamaan yang kurang jelas
Kesalahan penulisan
5. Setelah HTML, Lanjutkan ke CSS

Urutan belajar yang disarankan:

HTML
CSS
Flexbox
Grid
Responsive Design
JavaScript
Framework (Bootstrap/Tailwind)
